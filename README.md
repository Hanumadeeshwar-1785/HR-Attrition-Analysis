# HR Attrition Analysis Using IBM Employee Attrition Dataset

## Objective
Analyze employee attrition using preprocessing and exploratory data analysis techniques.

## Dataset
Final_dataset_Attrition.csv

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Tasks Performed
1. Data Preprocessing
2. Missing Value Handling
3. Duplicate Removal
4. Label Encoding
5. Feature Scaling
6. Exploratory Data Analysis
7. Correlation Analysis

## Results
The analysis identified important factors influencing employee attrition such as:
- Overtime
- Monthly Income
- Years At Company
- Job Role
- Department
